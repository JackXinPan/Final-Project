The Data

The key source of data was the National Notifiable Diseases Surveillance System: http://www9.health.gov.au/cda/source/rpt_2.cfm. This website can generate reports on notified cases by State, Year, Disease Grouping and Disease.

This data was selected as the Authors wish to investigate trends in infection rates between States and develop a tool that could guide strategic public health funding

Data was obtained from the site using a Python application developed by the Authors. The application utilises the Splinter and Pandas Libraries to scrape data from the website then the SQLAlchemy Libary to store this data in a SQLite Database.

An SQLAlchemy/Flask application was developed to create an API that could read data from the SQLite database into JavaScript

All charts were created and analysis was performed using the D3.js Libary. The charts are desgined to allow users to explore the dataset and compare infection rates by disease grouping, state, year and individual diseases.

This dataset could be extended by incorporating infection rates by age and sex and other contributing health statistics which would enable reconstructive analysis to identify key risk factors and establish early interventions.
